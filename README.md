# Evolved Snake

## Vercel Link

<https://evolved-snake-kdbh.vercel.app/>

## Design Concept

首先從操作上，除了原版的wasd控制方向，加入了空格鍵可以暫停，可以去思考下一步該怎麼走會更合理
並且因為多了鑽石跟炸彈，會更快加長蛇的長度，以及不能只是傻傻地繞著外圍就能避免死亡，希望玩家在玩的時候能更加謹慎地去選擇路線，也能讓遊戲變得更加刺激。

## How My Version Is Different

更多樣的食物，而且還會隨機生成炸彈

## How To Play

wasd控制方向，空格鍵暫停
